## 简介
**版本历史**

| 版本          | 日期          | 备注  |
| ------------  |:----------------:| ------------|
| v1.0.1        | 2019/4/3  |  初版  |

东润移动广告SDk(Android)是官方推出的移动广告SDK在Android平台上的版本（以下简称SDK）。

SDK的发行版本包括 AAR 包、Demo 示例、接入文档。

## 运行环境

可运行于`Android 4.0(API level 14)`及以上版本。

## 代码混淆
如果您需要使⽤ proguard 混淆代码，需确保不要混淆 SDK 的代码。
